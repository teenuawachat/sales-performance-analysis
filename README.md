# Sales Performance Analysis

## Objective
Analyze sales performance data to identify revenue trends, top-performing product categories, and high-value customers.

## Tools Used
- MySQL (Data Cleaning & Analysis)
- Power BI (Dashboarding)

## Key KPIs
- Total Revenue
- Revenue by Product Category
- Top Customers by Revenue
- Month-over-Month Growth

## Key Analysis
- Identified top revenue-generating product categories
- Analyzed customer contribution to overall revenue
- Evaluated month-over-month sales growth using window functions

## Dashboard
The Power BI dashboard provides interactive insights into revenue trends, category performance, and customer-level analysis.

## Outcome
Enabled stakeholders to track revenue performance and focus on high-impact products and customers.
